音樂撥放器

目的：設計並開發一個支持多種音樂格式的網頁音樂播放器，包括播放、暫停、音量調節和播放進度控制功能。

使用技術：這個項目使用 HTML5 的 `audio` 元素實現音樂播放，並結合 JavaScript 進行控制，如播放/暫停、音量調節和播放進度設定，CSS 用於設計用戶介面。

遇到問題：在開發過程中遇到了以下問題：

  • 自動播放功能無法取消：由於 `audio` 元素的 `autoplay` 屬性未能正確設置，導致播放器在頁面加載時自動播放音樂，無法由用戶控制。
  • 音量控制滑桿範圍不正確：音量控制滑桿的 `input` 事件處理函數未能正確映射音量範圍，導致音量設定不精確。
  • 播放進度條顯示不完全：播放進度條顯示邏輯中的 `seekTo` 函數未能正確計算播放進度，無法準確反映音樂的播放狀態。

解決方案：進行了以下修正：

  • 修正自動播放功能：移除了 `autoplay` 屬性，並在 `togglePlay` 函數中添加了播放和暫停的邏輯來控制音樂播放。
  • 調整音量控制：修正了音量滑桿的 `input` 事件處理函數，確保音量值正確映射到 `audio` 元素的 `volume` 屬性。
  • 更新播放進度條顯示：修正了 `seekTo` 函數的邏輯，確保播放進度條能夠正確顯示音樂的播放狀態。

經過上述調整後，播放器能夠順利運行並正確顯示音樂播放的狀態，實現了預期的功能。

  • 未顯示“加入購物車”按鈕：在商品詳細頁面中，“加入購物車”按鈕未能正確顯示。

  • 添加“加入購物車”按鈕：在商品詳細介紹區域加入“加入購物車”按鈕，並使用 JavaScript 處理按鈕點擊事件。

  • 完善購物車功能：實現了 `updateCart` 和 `removeFromCart` 函數來更新購物車內容，並提供了 `checkout` 結帳功能的提示。
  • 調整網站色調和排版：根據新的設計要求調整了 CSS 中的顏色和排版，確保網站整體風格一致。

結果：經過上述調整後，電子商務網站的功能和界面得到了改善，能夠更好地支持用戶的購物需求。

貪吃蛇遊戲 (C# 基礎開發)

功能：這款遊戲是使用 C# 開發的經典貪吃蛇遊戲，旨在學習基本的 C# 程式設計技能。玩家控制蛇的移動，吃取食物並增加長度，當蛇的長度超過限制或撞到自身時，遊戲結束。

  • 控制蛇的移動：程式碼允許玩家使用 WASD 鍵來改變蛇的移動方向。`direction` 變數用來決定蛇的移動路徑，確保蛇在正確的方向上移動。
  • 更新蛇的身體：在 `FixedUpdate` 方法中，蛇的每一節身體（除了頭部）會跟隨前一節的移動。這樣，蛇的身體會根據頭部的位置進行更新，形成一個連貫的蛇身。
  • 碰撞處理：當蛇的頭部碰到食物或障礙物時，程式會處理相應的事件。碰到食物時，會增加蛇身並更新分數；碰到障礙物時，會重置遊戲狀態並播放音樂，提示遊戲結束。
  • 重置遊戲：當遊戲結束時，蛇的位置會被重置到初始位置，身體會清空，分數也會重置。這樣可以讓玩家重新開始遊戲。

Food 類別功能：食物在遊戲中會在指定的區域內隨機生成位置。每當食物被吃掉時，食物的位置會重新隨機生成，保持遊戲的挑戰性和趣味性。

主要類別

  • `direction`：控制蛇的移動方向，根據玩家的輸入進行調整。
  • `speed`：控制遊戲的速度，通過調整時間縮放來實現。
  • `bodyPrefab`：蛇身體的預製件，用於在吃到食物後生成新的蛇身。
  • `bodies`：存儲蛇身所有節點的列表，包括頭部和各節身體。
  • `foodAera`：定義食物可以生成的區域範圍，確保食物在有效的範圍內生成。

結果：玩家控制蛇吃取食物並避免碰撞，遊戲提供了清晰的界面和流暢的操作體驗。遊戲的挑戰性和趣味性得到了有效提升。
